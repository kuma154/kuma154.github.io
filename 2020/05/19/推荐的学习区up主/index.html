推荐的一些B站up主(学习区)

学习区 前端教学
Bilibili UP主
发布日期:   2020-05-19

关于学习区，也是我发现B站这个新大陆的地方

学习区其实也分为很多分区
比如语言区，技术区，教学区等等
今天在这里我推荐的是我学习前端的一些宝藏区up主，虽然关注他们的粉丝不多，但视频都很精致，教学的很仔细，快来看看吧！

学习区

蜡笔和小勋


介绍：分享知识，感悟成长 I 微博@蜡笔和小勋 I 邮箱：labidakashe@163.com(蜡笔打卡社)

推荐视频：
av87954714→如何逼自己在家学习
av41654978→提升自律小技巧
av80872213→自动化的重要意义
av44360187→提升记忆小技巧
av71612834→如何养成背单词习惯

作为一个学习区的19年百大up，蜡笔和小勋无疑是一个奇特的存在，他们叫你的不是学习知识，而是教你如何去学习！
这些学习小技巧在我学习上还是记笔记上都给了我很大的帮助

峰华前端工程师


介绍：帮助开发者进步与成长/海归/博客：https://zxuqian.cn/群：644722908/关注我，你想要的全都有

6分钟学会Express 后端 API 开发 Node.js
给前端工程师看的后端科普
前后端交互之 HTTP 协议

峰华前端工程师的特点是将一些难懂的知识点配上精美的剪辑视频在几分钟的时间内给你讲述的明明白白
例如上面的链接和他讲述的grid以及flex布局等，他的视频剪辑无疑是他的一大特点，在前端道路上适合有些基础但细节不是太清楚的人学习

小陆前端


介绍：每日更新一条初级前端必备知识点

字节面经成功经验分享
前端面试题总结
HTTP协议总结

小陆前端是一位总结性up，并带给你很多的面试技巧以及面试题的总结，当你需要找工作或要复习的时候不妨来看看吧

小马哥_老师


介绍：大家好，我是小马哥老师，一位前端爱好者。希望这里的前端教学视频能给大家在职业生涯中增添光彩

Vue全家桶实战+Vue商城项目实战+如何部署上线+SSR服务器渲染
零基础小白必学的HTML
零基础学前端HTML之后必学的CSS
3小时Vue基础快速入门

小马哥_老师是一位非常有意思的up，很多小伙伴都说他的视频向看脱口秀一样，而且他讲述的知识点非常的细致，适合前端小白或零基础的人来学习

CodeSheep


介绍:一只程序猿up！IT咨询去微信公众号”CodeSheep”直接私信我

套路拆解:大厂是如何保证代码质量的
必读计算机编程好书推荐！程序员小伙搬出了他的书架！:本期视频给大家推荐自学计算机编程的一些经典好书吧，主题涉及：C语言/C++/Java/Python/Go语言等编程语言、数据结构和算法、计算机网络（TCP/IP协议栈）、Linux操作系统、数据库和SQL、Spring等Java EE企业级应用框架、重构和代码质量等几大方面。
包工头都能听明白的前后端分离！:在职程序员聊一聊互联网开发中火热的前后端分离。很多人一谈到这个问题就会钻到某项具体的技术中去，比如ajax异步请求、比如Spring Boot+Vue、比如RESTful接口和规范等等。本期视频想从另外一个角度来聊聊如何理解前后端分离、前后端分离在实际项目落地中有什么注意点以及前后端分离优缺点等几个话题。

CodeSheep是一位很特殊的up，他不是教你如何学习的up，是一位带你开拓互联网知识盲区的up，教你现如今该学习什么样的新技术，在你对为来迷茫的时候他的视频很适合你

技术胖


介绍：用心录制每一集视频教程，努力一年出100集免费视频教程。

教你变成疫情过后 最抢手的程序员:疫情过后，有四类程序员会遭到互联网公司的疯抢，这个视频就讲解如何变成复工后最抢手的程序员。
大学生如何作？才能轻松当高薪程序员:七个大学生最关心的程序员问题，解除你的困扰，让你加速成为最抢手的程序员
三招提高代码质量:作为程序员，你有没有遇到过下面三个情况?
1.你写的代码是不是被有经验的程序员嘲笑？
2.push上去的代码是不是测出很多Bug？
3.,Code Review时是不是让你无地自容？
如果你有上面三种情况，这个视频能解决你的烦恼。关注即可成为更优秀的程序员

技术胖是一位拥有十几年工作经验的前端工程师（看他的发亮也能看得出来），作为一个扫盲up，他帮你将未来的弯路告诉你，他的主页里也有许多系列教程想学习react等最新前端框架系列的人可以关注他，适合那些正在走向前端工程师路线以及基础较好的人观看

老张前端


介绍：微信：18511773322

老张新出课程，解决就业问题，实锤
老张开启第二期帮助找前端工作的朋友们完成就业
前端简历不会写咋办？已经找到工作的简历给你参考

老张前端是一位前端的面试官，他的视频会帮助你在找工作时如何将你的前端优势发挥出来，通过在线面试的方式带你扫除在面试过程中出现的问题
